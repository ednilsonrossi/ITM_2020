Essa é uma página de exemplo utilizada para mostrar o funcionamento do CSS em uma página HTML.

HTML

HTML foi criada para ser uma linguagem exclusivamente de marcação e estruturação de conteúdos.

Assim, não cabe ao HTML fornecer informações sobre a apresentação dos elementos, como por exemplo, cores de fontes, tamanho de texto, posicionamentos e todo aspecto visual de um documento.

CSS

CSS é a abreviação para o termo Cascading Style Sheet, traduzido como Folha de Estilo em Cascata.

Ao CSS cabe a função de apresentação visual de um documento na WEB. A formatação de nossas páginas é feita com essa linguagem.
